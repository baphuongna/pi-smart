# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.3.0] - 2026-05-13

### Added

- **Comprehensive Documentation** - README, API reference, Quick Start guide, Patterns reference
- **Research Patterns Applied** - Patterns from 52 source repositories including:
  - gstack (persistent daemon, design system, boil the lake)
  - gsd-2 (ADR system, branchless worktree, multi-model)
  - pi-crew (hooks, metrics, correlation, UI patterns)
  - beads (hash IDs, graph memory, memory decay)
  - context-mode (BM25 search, context sandbox)
  - everything-claude-code (quality gates, agent shield)
  - And more...

- **Build System** - TypeScript compilation with declaration files

### Features

- Full TypeScript support with declaration maps
- ESM modules with NodeNext resolution
- Comprehensive unit tests (tsx runner)
- Peer dependency on TypeScript 5.0+
